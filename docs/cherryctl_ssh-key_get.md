## cherryctl ssh-key get

Retrieves ssh-key details.

### Synopsis

Retrieves the details of the specified ssh-key.

```
cherryctl ssh-key get ID [flags]
```

### Examples

```
  # Gets the details of the specified ssh-key:
  cherryctl ssh-key get 12345
```
