## cherryctl team get

Retrieves team details.

### Synopsis

Retrieves the details of the specified team.

```
cherryctl team get ID [flags]
```

### Examples

```
  # Gets the details of the specified team:
  cherryctl team get 12345
```

### Options

```
  -h, --help          help for get
  -i, --team-id int   The team's ID.
```
